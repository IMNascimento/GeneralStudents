Para a utilização da função de filtro dinamica você deve seguir os passo a passo:
- crie um array informando os campos na sequencia que eles foram criados no html
o array deve estar passando o nome de todos os campos.
- crie um segundo array informando os tipos de dados que você está esperando em
cada campo tem que ser na sequencia igual foi feito com os campos.
- O resultado da função é um array já com dados corretos e filtrados. podendo ser
acessado por chave que é o nome do array e o nome do campo. Pois o array é associativo
ou a utilização da função extract para transforma as chaves em variaveis
